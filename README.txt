== Description ==

Oleggo LiveStream is a wordpress plugin that integrates video streaming, twitter and facebook to improve your streaming events.

Oleggo LiveStream can manage video streaming (from youtube, vimeo, ustream or whatever you want), plus you can add twitter hashtags search and facebook live streaming. Using these services you can create a great livestreaming event page.

== Installation ==

1. Upload the entire 'oleggo-livestream' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the LiveStream options in Oleggo LiveStream Settings. The control panel of Oleggo LiveStream is in 'Settings > Oleggo LiveStream'.
4. Place the text "[oleggo-livestream]" (without cuotes) inside a post or a page.
5. Edit your CSS for better display of the different divs.
5. Enjoy your LiveStream page. ;)

**Note: Be sure to have "wp\_head();" and "wp\_footer();" in your blog theme. If you don't, you can add it for let the plugin works.**
